# Queries
- **Disco - Distribuicao dos arquivos nos discos.sql**: Retorna distribuição dos arquivos de data e log nos discos, mostrando também o espaço livre
- **Disco - Espaco utilizado.sql**: Retorna discos que contem arquivos do SQL Server com espaço total e utilizado
- **Disco - Utilizacao de disco IOPS e Latencia.sql**: Retorna estatísticas de utilização de disco por arquivo, como IOPs e latência
- **Processos - requests em execucao.sql**: Retorna processos em execução, alternativa editavel à sp_who2 active e outras procedures semelhantes
